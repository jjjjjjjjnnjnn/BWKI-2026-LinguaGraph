# LinguaGraph Research Portal

> The unified entry point for the LinguaGraph project — findings, data, visualizations and paper.

## Quick Start

Open `index.html` in any browser. No build step required.

## Tech Stack

- Tailwind CSS v3 (CDN)
- Mermaid.js (architecture diagram)
- Chart.js (benchmark + coverage charts)
- Bootstrap Icons
- Vanilla JavaScript

## Deployment

GitHub Pages automatically deploys from `cognitive-space/web/`. The portal lives at `cognitive-space/portal/` and is accessible at:

```
https://<user>.github.io/BWKI-2026-LinguaGraph/portal/
```
