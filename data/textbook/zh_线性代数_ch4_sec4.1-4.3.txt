第四章 线性方程组

4.1 向量的线性相关性

一、线性组合

给定向量组 α₁, α₂, ..., αₘ，表达式

k₁α₁ + k₂α₂ + ... + kₘαₘ

称为这组向量的一个线性组合。

二、线性相关与线性无关

定义：如果存在不全为零的数 k₁, k₂, ..., kₘ 使得

k₁α₁ + k₂α₂ + ... + kₘαₘ = 0

则称向量组 α₁, α₂, ..., αₘ 线性相关，否则线性无关。

判定定理：
1. 含零向量的向量组一定线性相关
2. 两个向量线性相关 ⟺ 它们成比例
3. n+1 个 n 维向量一定线性相关

4.2 向量组的秩与极大线性无关组

一、极大线性无关组

向量组的一个部分组，满足：
1. 它本身线性无关
2. 向量组中任一向量都可由它线性表示

二、向量组的秩

向量组的极大线性无关组中所含向量的个数。

定理：矩阵的行秩 = 矩阵的列秩 = 矩阵的秩

4.3 线性方程组解的结构

一、齐次线性方程组

Ax = 0 的解构成一个向量空间（解空间）。
基础解系：解空间的基，含有 n - r(A) 个向量。

二、非齐次线性方程组

Ax = b 的通解 = 特解 + 齐次方程组的通解

定理：非齐次方程组有解的充分必要条件是 r(A) = r(A|b)

线性相关性与矩阵的关系：线性相关性决定了矩阵的秩，而矩阵的秩决定了线性方程组解的结构。
