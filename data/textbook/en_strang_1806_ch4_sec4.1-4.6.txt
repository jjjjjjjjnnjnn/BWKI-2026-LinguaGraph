Chapter 4 Vector Spaces

4.1 Vectors in Rⁿ

A vector in Rⁿ is an ordered n-tuple v = (v₁, v₂, ..., vₙ).

Operations:
- Addition: u + v = (u₁+v₁, ..., uₙ+vₙ)
- Scalar multiplication: cv = (cv₁, ..., cvₙ)

4.2 Linear Combinations and Span

A vector w is a linear combination of v₁, ..., vₖ if w = c₁v₁ + ... + cₖvₖ.

The span of {v₁, ..., vₖ} is the set of all linear combinations.

4.3 Linear Independence

Vectors v₁, ..., vₖ are linearly independent if c₁v₁ + ... + cₖvₖ = 0 implies c₁ = ... = cₖ = 0.

4.4 Basis and Dimension

A basis for V is a linearly independent spanning set.
The dimension dim(V) is the number of vectors in any basis.

Key theorem: Any two bases for a finite-dimensional vector space have the same number of vectors.

4.5 Change of Basis

If B = {v₁, ..., vₙ} and B' = {w₁, ..., wₙ} are two bases, the transition matrix P satisfies [x]_B' = P⁻¹[x]_B.

4.6 Rank and Nullity

Rank(A) = dimension of column space = dimension of row space
Nullity(A) = dimension of null space
Rank-Nullity Theorem: rank(A) + nullity(A) = n

Key relationship: Vector spaces provide the abstract framework for understanding linear equations, matrices, and linear transformations. Basis and dimension are the fundamental invariants.
