3.2 高阶线性微分方程

一、n 阶线性微分方程的一般形式

y⁽ⁿ⁾ + a₁(x)y⁽ⁿ⁻¹⁾ + ... + aₙ(x)y = f(x)

齐次：f(x) = 0；非齐次：f(x) ≠ 0

二、齐次方程的解的结构

定理：若 y₁, y₂, ..., yₙ 是齐次方程的 n 个线性无关的解，则

y = C₁y₁ + C₂y₂ + ... + Cₙyₙ

是齐次方程的通解。

三、非齐次方程的解的结构

非齐次通解 = 齐次通解 + 非齐次特解

3.3 常系数齐次线性微分方程

一、二阶常系数齐次方程

y'' + py' + qy = 0

特征方程：r² + pr + q = 0

解的三种情况：
1. Δ > 0：两个不等实根 r₁, r₂ → y = C₁e^(r₁x) + C₂e^(r₂x)
2. Δ = 0：重根 r → y = (C₁ + C₂x)e^(rx)
3. Δ < 0：共轭复根 α ± βi → y = e^(αx)(C₁cos βx + C₂sin βx)

二、推广到 n 阶

特征方程有 n 个根（含重根和复根），按规则写出通解。

3.4 常系数非齐次线性微分方程

一、待定系数法

f(x) = Pₘ(x)e^(λx) 型：

设特解 y* = x^k Qₘ(x)e^(λx)，其中 k 是 λ 作为特征根的重数

二、参数变易法

已知齐次通解 y = C₁y₁ + C₂y₂，设 y* = u₁(x)y₁ + u₂(x)y₂

解方程组求 u₁', u₂'

微分方程与导数的关系：微分方程的核心是导数——方程中包含未知函数的导数，求解过程就是从导数反推原函数。
