第十章 偏微分方程基础

10.1 三类经典偏微分方程

一、波动方程

∂²u/∂t² = c²∂²u/∂x²

描述：弦振动、声波、电磁波
达朗贝尔解：u(x,t) = f(x-ct) + g(x+ct)

二、热传导方程

∂u/∂t = k∂²u/∂x²

描述：温度分布、扩散过程
解的特点：光滑化效应，t > 0 时解立即变光滑

三、拉普拉斯方程

∂²u/∂x² + ∂²u/∂y² = 0

描述：稳态温度、电势、流速势
调和函数：满足拉普拉斯方程的函数

10.2 分离变量法

一、基本思想

假设 u(x,t) = X(x)·T(t)，代入 PDE 后分离变量。

二、例：波动方程的初边值问题

u_tt = c²u_xx, u(0,t) = u(L,t) = 0, u(x,0) = φ(x)

分离变量得：X'' + λX = 0, T'' + c²λT = 0

特征值问题：X_n(x) = sin(nπx/L), λ_n = (nπ/L)²

级数解：u(x,t) = Σ Bₙ sin(nπx/L) cos(nπct/L)

10.3 傅里叶级数

一、傅里叶级数展开

f(x) = a₀/2 + Σ[aₙcos(nπx/L) + bₙsin(nπx/L)]

其中 aₙ = (1/L)∫f(x)cos(nπx/L)dx, bₙ = (1/L)∫f(x)sin(nπx/L)dx

二、Parseval 等式

(1/L)∫|f(x)|²dx = a₀²/2 + Σ(aₙ² + bₙ²)

三、收敛性

Dirichlet 条件：分段光滑函数的傅里叶级数收敛

偏微分方程与傅里叶分析的关系：分离变量法产生特征值问题，其解构成傅里叶级数——傅里叶分析是求解 PDE 的核心工具。
