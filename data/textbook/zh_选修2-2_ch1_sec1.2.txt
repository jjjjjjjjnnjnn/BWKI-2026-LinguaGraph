1.2 基本初等函数的导数公式与导数运算法则

一、基本初等函数的导数公式

我们已经学习了几个基本初等函数的导数：

1. 常数函数的导数：(C)' = 0
2. 幂函数的导数：(x^n)' = nx^(n-1)
3. 正弦函数的导数：(sin x)' = cos x
4. 余弦函数的导数：(cos x)' = -sin x
5. 指数函数的导数：(a^x)' = a^x · ln a，特别地 (e^x)' = e^x
6. 对数函数的导数：(log_a x)' = 1/(x · ln a)，特别地 (ln x)' = 1/x

二、导数的四则运算法则

设函数 u(x) 和 v(x) 都可导，则：

1. 和差法则：[u(x) ± v(x)]' = u'(x) ± v'(x)
2. 积法则：[u(x) · v(x)]' = u'(x)·v(x) + u(x)·v'(x)
3. 商法则：[u(x)/v(x)]' = [u'(x)·v(x) - u(x)·v'(x)] / [v(x)]²，其中 v(x) ≠ 0

积法则可以推广到多个函数相乘的情况。例如：
(uvw)' = u'vw + uv'w + uvm'

三、应用举例

例1：求 y = x³ + 2sin x 的导数。
解：y' = 3x² + 2cos x

例2：求 y = x² · e^x 的导数。
解：由积法则，y' = 2x · e^x + x² · e^x = e^x(2x + x²)

例3：求 y = sin x / x 的导数。
解：由商法则，y' = (cos x · x - sin x · 1) / x² = (x cos x - sin x) / x²

四、导数运算法则的意义

导数的运算法则使我们能够对复杂的函数进行求导，而不需要每次都回到导数的定义（极限）去计算。这些法则构成了微分学的计算基础。

注意：导数的四则运算法则建立在极限运算法则的基础之上。和差法则本质上是极限的加法法则的直接推论。
