第二章 定积分及其应用

2.1 定积分的概念

一、引例：曲边梯形的面积

问题：求由曲线 y = f(x)（f(x) ≥ 0）、x 轴以及直线 x = a、x = b 所围成的曲边梯形的面积。

解决思路：
1. 分割：将区间 [a, b] 等分成 n 个小区间，每个小区间长度 Δx = (b-a)/n
2. 近似：在每个小区间上，用矩形面积近似代替曲边梯形面积
3. 求和：将所有矩形面积相加，得到面积的近似值
4. 取极限：令 n → ∞（即 Δx → 0），得到面积的精确值

这个极限过程产生了定积分的概念。

二、定积分的定义

设函数 f(x) 在区间 [a, b] 上有界。将 [a, b] 任意分成 n 个小区间，在每个小区间上任取一点 ξ_i，作和式：

S_n = Σ f(ξ_i) · Δx_i

如果当最长小区间的长度趋于零时，S_n 的极限存在且与分法和取法无关，则称此极限为 f(x) 在 [a, b] 上的定积分，记作：

∫_a^b f(x) dx

其中 a 和 b 称为积分下限和积分上限，f(x) 称为被积函数，x 称为积分变量。

三、定积分的几何意义

1. 当 f(x) ≥ 0 时，∫_a^b f(x) dx 表示曲边梯形的面积
2. 当 f(x) ≤ 0 时，∫_a^b f(x) dx = -（曲边梯形的面积）
3. 一般情况下，∫_a^b f(x) dx 表示 x 轴上方的面积减去 x 轴下方的面积

四、定积分的基本性质

1. 线性性质：∫[af(x) + bg(x)]dx = a∫f(x)dx + b∫g(x)dx
2. 区间可加性：∫_a^b f(x)dx = ∫_a^c f(x)dx + ∫_c^b f(x)dx
3. 比较性质：若 f(x) ≤ g(x)，则 ∫_a^b f(x)dx ≤ ∫_a^b g(x)dx
4. 估值定理：若 m ≤ f(x) ≤ M，则 m(b-a) ≤ ∫_a^b f(x)dx ≤ M(b-a)

五、微积分基本定理（牛顿-莱布尼茨公式）

定理：设 f(x) 在 [a, b] 上连续，F(x) 是 f(x) 的一个原函数（即 F'(x) = f(x)），则：

∫_a^b f(x) dx = F(b) - F(a)

这个定理揭示了定积分与不定积分（原函数）之间的深刻联系，是微积分学的核心定理。它将定积分的计算转化为求原函数的问题。
