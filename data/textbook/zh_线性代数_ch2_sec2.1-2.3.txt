第二章 矩阵及其运算

2.1 矩阵的基本概念

一、矩阵的定义

由 m×n 个数按一定次序排成的 m 行 n 列的数表称为 m×n 矩阵。

特殊矩阵：
- 方阵：m = n 的矩阵
- 单位矩阵 I：主对角线为 1，其余为 0 的方阵
- 零矩阵：所有元素为 0 的矩阵
- 对角矩阵：非对角元素全为 0 的方阵
- 对称矩阵：A^T = A 的方阵

二、矩阵的转置

将矩阵的行与列互换：(A^T)ᵢⱼ = Aⱼᵢ

转置的性质：
1. (A^T)^T = A
2. (A+B)^T = A^T + B^T
3. (kA)^T = kA^T
4. (AB)^T = B^T A^T

2.2 矩阵的运算

一、矩阵加法

A + B = (aᵢⱼ + bᵢⱼ)

二、矩阵数乘

kA = (kaᵢⱼ)

三、矩阵乘法

C = AB，其中 cᵢⱼ = Σ aᵢₖ · bₖⱼ

注意：矩阵乘法不满足交换律，即 AB ≠ BA（一般情况下）

四、方阵的幂

Aⁿ = A · A · ... · A（n 次）

2.3 逆矩阵

定义：若 AB = BA = I，则 B 称为 A 的逆矩阵，记作 A⁻¹。

定理：方阵 A 可逆的充分必要条件是 |A| ≠ 0。

求逆公式：A⁻¹ = adj(A) / |A|，其中 adj(A) 是 A 的伴随矩阵。

逆矩阵的性质：
1. (A⁻¹)⁻¹ = A
2. (AB)⁻¹ = B⁻¹A⁻¹
3. (A^T)⁻¹ = (A⁻¹)^T

矩阵与线性方程组的关系：矩阵是线性方程组的紧凑表示，Ax = b 中 A 是系数矩阵，x 是未知数向量，b 是常数项向量。矩阵的可逆性决定了方程组是否有唯一解。
