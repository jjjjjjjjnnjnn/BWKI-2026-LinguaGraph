1.5 泰勒公式与泰勒展开

一、泰勒公式的动机

问题：能否用多项式函数来近似一个复杂的可导函数？

基本思想：如果 f(x) 在 x = a 处有 n 阶导数，则可以用一个 n 次多项式来近似 f(x)，使得在 x = a 附近，多项式与 f(x) 的值及其各阶导数值都相等。

二、泰勒公式

定理（泰勒公式）：设 f(x) 在 x = a 的某邻域内有 n+1 阶导数，则对邻域内任意 x，存在介于 a 和 x 之间的 ξ，使得：

f(x) = f(a) + f'(a)(x-a) + f''(a)(x-a)²/2! + ... + f⁽ⁿ⁾(a)(x-a)ⁿ/n! + R_n(x)

其中 R_n(x) = f⁽ⁿ⁺¹⁾(ξ)(x-a)ⁿ⁺¹/(n+1)! 称为拉格朗日余项。

三、麦克劳林公式

当 a = 0 时，泰勒公式变为麦克劳林公式：

f(x) = f(0) + f'(0)x + f''(0)x²/2! + ... + f⁽ⁿ⁾(0)xⁿ/n! + R_n(x)

常用函数的麦克劳林展开：
- e^x = 1 + x + x²/2! + x³/3! + ... = Σxⁿ/n!
- sin x = x - x³/3! + x⁵/5! - ... = Σ(-1)ⁿx²ⁿ⁺¹/(2n+1)!
- cos x = 1 - x²/2! + x⁴/4! - ... = Σ(-1)ⁿx²ⁿ/(2n)!
- ln(1+x) = x - x²/2 + x³/3 - ... = Σ(-1)ⁿ⁻¹xⁿ/n

四、泰勒公式的应用

1. 近似计算：用有限项多项式近似计算函数值
2. 求极限：将函数展开后比较阶数
3. 证明不等式：利用余项的符号
4. 级数理论：泰勒级数的收敛性

泰勒公式与导数的关系：泰勒公式是导数概念的推广——它不仅利用了一阶导数（切线），还利用了高阶导数来提供更精确的近似。
