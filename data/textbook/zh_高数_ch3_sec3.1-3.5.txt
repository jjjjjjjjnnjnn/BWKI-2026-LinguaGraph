同济大学《高等数学》第七版 — 第三章 微分中值定理与导数的应用

3.1 微分中值定理

一、罗尔定理

若 f(x) 在 [a,b] 连续，(a,b) 可导，且 f(a) = f(b)，则 ∃ξ ∈ (a,b) 使 f'(ξ) = 0。

二、拉格朗日中值定理

f(b) - f(a) = f'(ξ)(b - a)

三、柯西中值定理

[f(b)-f(a)]/[g(b)-g(a)] = f'(ξ)/g'(ξ)

3.2 洛必达法则

0/0 型和 ∞/∞ 型未定式的极限。

3.3 泰勒公式

f(x) = Σf⁽ᵏ⁾(a)(x-a)ᵏ/k! + Rₙ(x)

3.4 函数的单调性与极值

一、单调性判定

f'(x) > 0 → 递增；f'(x) < 0 → 递减

二、极值

f'(x₀) = 0 且 f''(x₀) < 0 → 极大值
f'(x₀) = 0 且 f''(x₀) > 0 → 极小值

3.5 曲线的凹凸性与拐点

f''(x) > 0 → 凹；f''(x) < 0 → 凸

中值定理是导数应用的理论基础：它将函数值的差与导数联系起来。
