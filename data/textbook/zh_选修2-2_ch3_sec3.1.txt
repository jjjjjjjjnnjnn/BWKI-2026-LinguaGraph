第三章 微分方程简介

3.1 微分方程的基本概念

一、微分方程的定义

含有未知函数及其导数（或微分）的方程称为微分方程。未知函数是一元函数的微分方程称为常微分方程（ODE）。

一般形式：F(x, y, y', y'', ..., y⁽ⁿ⁾) = 0

其中最高阶导数的阶数称为微分方程的阶。

二、微分方程的解

如果将函数 y = φ(x) 代入微分方程后使方程成为恒等式，则称 y = φ(x) 为该微分方程的解。

通解：含有与微分方程阶数相同个数的任意常数的解。
特解：给任意常数以确定值后得到的解。
初始条件：确定特解所需的条件，如 y(x₀) = y₀, y'(x₀) = y₀' 等。

三、一阶微分方程

可分离变量方程：dy/dx = f(x)g(y)
解法：∫dy/g(y) = ∫f(x)dx

齐次方程：dy/dx = φ(y/x)
解法：令 u = y/x，转化为可分离变量方程

一阶线性方程：dy/dx + P(x)y = Q(x)
解法：y = e^(-∫Pdx)[∫Qe^(∫Pdx)dx + C]

四、应用举例

例1：求微分方程 dy/dx = 2x 的通解。
解：y = ∫2x dx = x² + C

例2：求微分方程 dy/dx = y，y(0) = 1 的特解。
解：分离变量得 dy/y = dx，积分得 ln|y| = x + C
由 y(0) = 1 得 C = 0，所以 y = e^x

例3：放射性衰变模型 dN/dt = -λN，解为 N = N₀e^(-λt)

微分方程与导数的关系：微分方程是导数概念的直接应用——它描述了函数与其变化率之间的关系。
