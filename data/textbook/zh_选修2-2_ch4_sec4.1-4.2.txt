第四章 无穷级数

4.1 常数项级数的概念与性质

一、级数的定义

给定数列 {uₙ}，表达式

Σ(n=1→∞) uₙ = u₁ + u₂ + u₃ + ...

称为无穷级数（常数项级数）。部分和 Sₙ = u₁ + u₂ + ... + uₙ。

二、级数的收敛与发散

如果部分和数列 {Sₙ} 有极限 S，则称级数收敛，S 称为级数的和。
如果 {Sₙ} 没有极限，则称级数发散。

三、级数的基本性质

1. 级数收敛的必要条件：若 Σuₙ 收敛，则 lim uₙ = 0
2. 线性性质：若 Σaₙ 和 Σbₙ 都收敛，则 Σ(αaₙ + βbₙ) 也收敛
3. 在级数前面增加或去掉有限项，不影响级数的收敛性
4. 收敛级数可以任意加括号，仍然收敛

四、正项级数的判别法

1. 比较判别法：若 0 ≤ uₙ ≤ vₙ，且 Σvₙ 收敛，则 Σuₙ 收敛
2. 比值判别法：若 lim uₙ₊₁/uₙ = ρ，则 ρ < 1 时收敛，ρ > 1 时发散
3. 根值判别法：若 lim ⁿ√uₙ = ρ，则 ρ < 1 时收敛，ρ > 1 时发散

4.2 幂级数

一、幂级数的定义

形如 Σ(n=0→∞) aₙ(x-a)ⁿ 的级数称为幂级数。当 a = 0 时为 Σaₙxⁿ。

二、收敛半径

定理（阿贝尔定理）：如果幂级数 Σaₙxⁿ 在 x = x₀ ≠ 0 处收敛，则在 |x| < |x₀| 时绝对收敛；在 x = x₁ 处发散，则在 |x| > |x₁| 时发散。

收敛半径 R = 1/lim |aₙ₊₁/aₙ|（比值法）

三、幂级数的运算

在收敛区间内，幂级数可以逐项求导、逐项积分。

幂级数与泰勒级数的关系：泰勒级数就是幂级数——将函数在某点展开为幂级数。收敛半径决定了泰勒级数的有效范围。
