4.1 不定积分的概念与性质

一、原函数

若 F'(x) = f(x)，则 F(x) 是 f(x) 的一个原函数。

二、不定积分

∫f(x)dx = F(x) + C

三、基本积分表

∫xⁿdx = xⁿ⁺¹/(n+1) + C
∫(1/x)dx = ln|x| + C
∫eˣdx = eˣ + C
∫sin x dx = -cos x + C
∫cos x dx = sin x + C

4.2 换元积分法

一、第一类换元（凑微分）

∫f(g(x))g'(x)dx = ∫f(u)du

二、第二类换元

令 x = ψ(t)，dx = ψ'(t)dt

4.3 分部积分法

∫u dv = uv - ∫v du

选取原则：反对幂三指（反三角、对数、幂函数、三角、指数）
