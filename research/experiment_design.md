# LinguaGraph — BWKI 2026 Formal Experiment Design

## 1. Research Overview

### Core Research Question
> Does language shape thinking? Can AI measure this change?

### Experimental Design
| Dimension | Choice | Rationale |
|-----------|--------|-----------|
| Design type | **Within-subject + Between-subject mixed** | Within: same person answers in 3 languages; Between: compare native groups |
| Independent variable | Language (ZH / DE / EN) | 3 levels |
| Dependent variables | LDS, Concept Set, Graph Structure, Concept Centrality | 4 metrics per participant |
| Participants | 30 (10 ZH-native, 10 DE-native, 10 EN-native) | Balanced trilingual sample |
| Materials | 5 social topics × 3 languages = 15 questions | Open-ended, 100-300 words each |

### Hypothesis Structure
```
H₀: Language has NO effect on cognitive structure
    (same concepts and relations regardless of language)
H₁: Language DOES affect cognitive structure
    (systematically different concept networks per language)
```

### Power Analysis
- Effect size expectation: d = 0.6–0.8 (based on pilot LDS range 0.61–1.00)
- Required: 10 per group (within-subject, paired) → power > 0.80 at α = 0.05
- Target: 30 total (15 per group for between-subject comparisons)

## 4. Confounding Variables & Controls

### 4.1 Identified Confounders

| Confounder | Risk | Control Strategy |
|------------|------|------------------|
| **Language proficiency** | Uneven skill → shorter answers in weaker language | Screen for B2+ in all 3 languages; measure self-reported proficiency |
| **Order effects** | First language answers may prime subsequent ones | Randomize language order across participants (Latin square) |
| **Topic familiarity** | Some topics may be more discussed in one culture | Ask "how often do you discuss X" as covariate |
| **Social desirability** | Participants may give culturally expected answers | Anonymous survey; emphasize "no wrong answers" |
| **Translation effect** | Words like 自由/Freiheit are not exact equivalents | Include "untranslatable concept" post-survey reflection |
| **Time pressure** | Longer answers = richer graphs | No time limit; encourage 3-5 sentences |
| **Recruitment bias** | WeChat-based sampling favors certain demographics | Also recruit through university mailing lists |

### 4.2 Control Questions

Include 2 control questions that should produce LOW drift:
- "What is food?" / "Was ist Essen?" / "什么是食物？"
- "What is weather?" / "Was ist Wetter?" / "什么是天气？"

Expected: LDS < 0.4 (universal concepts). If control LDS > 0.6 → method failure.

### 4.3 Counterbalancing

```
Participant 1: ZH → DE → EN
Participant 2: DE → EN → ZH
Participant 3: EN → ZH → DE
Participant 4: ZH → EN → DE
... (6 permutations, cycle through)
```

## 5. Analysis Plan

### 5.1 Pipeline

```
Raw Answers (ZH/DE/EN)
    ↓
LLM Concept Extraction (extract.py)
    ↓
Graph Construction (graph.py) — co-occurrence method
    ↓
Cross-Language Comparison (scoring.py)
    ┣━ LDS per topic per participant
    ┣━ Concept Shift: set differences per language pair
    ┣━ Centrality Shift: rank-order change of top concepts
    ┗━ Aggregate LDS per participant (average across 5 topics)
    ↓
Statistical Testing
    ┣━ H₁ test: One-sample t-test (LDS > 0.5?)
    ┣━ H₂ test: Paired t-test (within-subject LDS by language pair)
    ┣━ H₃ test: ANOVA (LDS × native language group)
    ┗━ Post-hoc: Tukey HSD for pairwise comparisons
```

### 5.2 Success Criteria (BWKI submission)

| Criterion | Target | Measurement |
|-----------|--------|-------------|
| Concept extraction F1 | ≥ 0.80 | Compare LLM vs human annotation |
| LDS > 0.5 for at least 3/5 topics | ≥ 3 topics | One-tailed t-test per topic |
| Within-subject LDS consistency | ICC > 0.6 | Intraclass correlation |
| Control topic LDS | < 0.5 | Should be lower than experimental |
| Between-group difference | p < 0.05 | ANOVA by native language |

### 5.3 Sample Code for Analysis

```python
# After all data is collected and processed:

import numpy as np

# H1: Is mean LDS significantly > 0.5?
t_stat, p_val = stats.ttest_1samp(lds_scores, 0.5)

    lds_zh_de, lds_zh_en, lds_de_en
```
