# LinguaGraph 招募操作手册（傻瓜版）

> 你只需要做一件事：把链接发出去。
> 调查链接：`https://jjjjjjjjnnjnn.github.io/BWKI-2026-LinguaGraph/survey/`

---

## 渠道一：小红书（你已有的）

### 步骤

1. 打开小红书 APP
2. 点底部 + 号发笔记
3. 标题写：
   ```
   🔬 3种语言的人怎么理解「成功」和「自由」？求中文母语者帮忙！
   ```
4. 正文直接复制这一段：

   ```
   大家好！我在做一个关于「语言和思维方式」的研究项目（德国BWKI 2026 AI竞赛）。
   
   需要中文母语者帮忙填一份问卷，非常简单：
   ✅ 用你的母语回答5个开放式问题（每题写3-5句话就行）
   ✅ 全程匿名，不收集任何个人信息
   ✅ 只需要10-15分钟
   ✅ 任何设备都能打开
   
   你的回答会直接帮助我理解：语言是否影响我们思考方式。
   
   问卷链接在评论区！或者直接点这里：
   https://jjjjjjjjnnjnn.github.io/BWKI-2026-LinguaGraph/survey/
   
   提前感谢每一个帮忙的人！🙏
   ```

5. **重要**：发出后你自己在评论区发一次链接（小红书正文里不能点，评论区可以）
6. 加上话题标签：`#问卷调查 #学术研究 #心理学 #语言 #跨文化 #留学 #德国留学 #BWKI`

---

## 渠道二：微信（最有效的中文渠道）

### 步骤

1. 打开微信
2. 打开任意群聊（同学群、家庭群、兴趣群）
3. 发送以下消息：

   ```
   【求助】在做一个人工智能竞赛项目，需要中文母语者帮忙填一份问卷 🧠
   
   只需要10-15分钟，用中文回答5个开放式问题。
   完全匿名，不收集任何个人信息。
   
   问卷链接：https://jjjjjjjjnnjnn.github.io/BWKI-2026-LinguaGraph/survey/
   
   这是我自己的研究项目，帮助很大！提前谢谢大家 🙏
   ```

4. 也可以发朋友圈：
   - 点「发现」→「朋友圈」→ 长按右上角相机图标
   - 粘贴上面那段话
   - 点击「发表」

---

## 渠道三：学校（Schloss Heessen — 德文渠道）

你可以在学校内部发德文版：

### 步骤

1. 找班主任或 school admin
2. 跟他们说："我在做一个德国AI竞赛项目（BWKI 2026），需要德语母语者填一份匿名问卷，能不能帮我发到学校群里？"
3. 给他们这段德文文案：

   ```
   📢 Umfrage für mein BWKI 2026 Forschungsprojekt
   
   Hallo zusammen! Ich bin Schüler am Schloss Heessen und forsche im Rahmen des Bundeswettbewerbs Künstliche Intelligenz 2026.
   
   Ich suche deutsche Muttersprachler für eine kurze Umfrage:
   ✅ 5 offene Fragen auf Deutsch beantworten (je 3-5 Sätze)
   ✅ ~10-15 Minuten
   ✅ Anonym, keine persönlichen Daten
   ✅ Geht von jedem Gerät
   
   Link: https://jjjjjjjjnnjnn.github.io/BWKI-2026-LinguaGraph/survey/
   
   Vielen Dank für eure Hilfe! 🙏
   ```

4. 如果学校有WhatsApp群/Telegram群/Schulchat，直接发进去

## 渠道四：Reddit r/SampleSize（英文渠道）

这是 Reddit 上专门给人填问卷的板块，效果很好。

### 步骤

1. 打开浏览器，访问 https://www.reddit.com/r/SampleSize/
2. 点右上角「Create Post」
3. 标题写：
   ```
   [Academic] Language and thought: How do you define success, freedom, justice? (ZH/DE/EN native speakers, 10-15 min)
   ```
4. 正文写：

   ```
   Hi everyone! I'm a student researcher working on a cross-linguistic cognitive science project for the BWKI 2026 AI competition in Germany.
   
   I need native speakers of Chinese, German, or English (age 14+) to answer 5 open-ended questions in their native language.
   
   Link: https://jjjjjjjjnnjnn.github.io/BWKI-2026-LinguaGraph/survey/
   
   - Takes 10-15 minutes
   - Fully anonymous
   - No personal data collected
   - Works on any device
   
   Thank you so much! Your responses will help us understand whether language shapes how we organize social concepts.
   ```
5. 加上 Flair：`[Academic]`
6. 点击「Post」

## 渠道五：知乎（中文渠道备用）

### 步骤

1. 打开知乎
2. 搜索「问卷调查」或「语言和思维」
3. 找一个相关问题，写回答，末尾加问卷链接
4. 或者直接发想法：
   - 点右下角 + → 想法
   - 粘贴小红书文案的简短版

---

## 渠道六：大学邮件（德文—HSHL）

联系你顾问所在的 Hamm-Lippstadt Hochschule：

1. 问你的 HSHL 顾问："能不能帮我把这份招募发到学校邮件列表？"
2. 给他们德文文案（见渠道三）

## 追踪表格

每发完一个渠道，在 `docs/recruitment_plan.md` 的 Tracking 表里记一笔：

```

```

## 接下来的具体操作顺序

```
第一天：
① 小红书发笔记 → 2分钟
② 微信发朋友圈 + 发3个群 → 2分钟
③ 学校找班主任/发群 → 5分钟

第二天：
④ Reddit r/SampleSize 发帖 → 3分钟

一周后：
⑤ 看哪些渠道有效 → 重复有效渠道
⑥ 如果不够 → 知乎 + 大学邮件
```
