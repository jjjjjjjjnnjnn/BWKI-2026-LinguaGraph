# Identifying Elements Essential for BERT's Multilinguality

**Source:** https://aclanthology.org/2020.emnlp-main.358/
**Crawled:** 2026-06-17

It has been shown thatmultilingualBERT(mBERT) yields high qualitymultilingualrepresentations and enables effective zero-shot transfer. This is surprising given that mBERT does not use any crosslingual signal during training. While recent literature has studied this phenomenon, the reasons for the multilinguality are still somewhat obscure.
