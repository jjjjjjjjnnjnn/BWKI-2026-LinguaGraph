RQ1: Language

Question: How do different languages organize the same knowledge?

ZH–DE textbooks show highest structural divergence; ZH–EN the lowest, suggesting curriculum tradition (not language family) drives organization.
LDS: 0.802–0.907
🔬

RQ2: Discipline

Question: How do different subjects organize their knowledge?

Knowledge density peaks early (middle/elementary) then declines. Depth is bounded at HDS ≤ 8. Physics has 2.1× more sequential depth than math.
CDS: 0.042–0.271
📋

RQ3: Education System

Question: How do different education systems organize curricula?

Coverage varies dramatically: UK 82% → US 76% → NRW 34% → CN 8%. Trajectories reveal system design philosophy (exam-driven vs. specialization).
CS: 8%–82%

Methods

Four metrics + gold standard validation

CDS
Concept Density Score
2|E| / (|V|·(|V|−1))
How densely connected knowledge is at each education level. Higher = more integrated.
HDS
Hierarchy Depth Score
max BFS depth(prerequisite)
Longest prerequisite chain for each concept. Measures knowledge hierarchy depth.
LDS
Language Drift Score
1 − mean(GED, Jaccard)
Cross-language structural divergence. 0 = identical, 1 = completely different.
CS
Coverage Score
|V_tb ∩ V_curr| / |V_curr|
How much of the curriculum the textbooks cover. Textbook–curriculum alignment.

Phase 0 — Pilot

Concept & Feasibility Study

3-student pilot study, 20 gold labels, initial LDS analysis across ZH/EN/DE. Validated the core hypothesis that cross-language knowledge structures measurably differ.

Phase 1 — Mathematics

Mathematics Knowledge Graph (574 concepts, 3,538 relations)

68 textbooks across ZH/EN/DE. MIMO extraction pipeline. First CDS/HDS/LDS measurements. Discovered peak-and-decline density pattern (F1–F5).

Phase 2 — Physics

Physics Knowledge Graph (366 concepts, 383 relations)

94 textbook versions across 3 languages. Cross-validated the CDS peak pattern. Found physics peaks at Elementary vs Math at Middle (F6). Depth is 2.1× Math (F7).

Phase 3 — Chemistry

Chemistry Knowledge Graph (220 concepts, 215 relations)

18 textbook versions. Confirmed universal STEM density pattern (F8). Three-discipline comparison complete.

Phase 4 — Curriculum

Curriculum Integration (4 education systems)

Aligned all three knowledge graphs against NRW, UK, US, and CN curricula. Built the Coverage Score metric. Discovered dramatic coverage differences (F9/F10).

Phase 5 — Coverage Score

Cross-System Coverage Analysis

Deep analysis of coverage trajectories. Found system design philosophy drives coverage patterns (exam-driven vs specialization). Three competing explanations evaluated.

Phase 6 — Validation

Gold Standard Validation & Model Benchmark

Expanded from 20 to 92 gold labels. Benchmarked 20 models across Bailian API. qwen-plus selected as production model (F1=0.939 social, F1=0.882 overall).

Phase 7 — Publication

Paper, Release & GitHub Professionalization

Full paper with 3 conclusions, methodology, results, discussion, error analysis. Release v0.2. GitHub Pages deployment. Trilingual READMEs. Comprehensive references.
