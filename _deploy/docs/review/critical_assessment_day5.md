## 3. 最大风险（我之前在回避这个）

你说"240 份真实问卷 5 天后到"。这很好，但有一个我没问的问题：

> **这 240 份问卷来自中学生，回答是自由文本——LLM 提取的质量能保证吗？**

这是 Pipeline A 最大的风险点。如果提取质量不够好（concept F1 < 0.70），所有下游 LDS/cross-language analysis 的结果都会受质疑。

`extractions` 表里有 57 条记录，但这是 pilot 数据。大规模跑 240 人 × 5 题 × 3 语 = 3600 条提取任务，需要：

1. **提取质量验证**：20 条 gold label 对比，当前 F1 多少？
2. **成本估算**：3600 次 LLM 调用，GPT-4.1-mini 或 Qwen3-8B，总 token 量？
3. **时间估算**：串行 vs 并行，批次多大？
