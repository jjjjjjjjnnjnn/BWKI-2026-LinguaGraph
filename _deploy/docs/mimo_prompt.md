### 背景

数学知识图谱已达到 287 概念，但**全部集中在高中和大学**：

```
当前分布:
  high:    99 概念  ✅
```

### 必填的概念链（全部学段）

以下每条链都必须完整提取，缺失任一个都会导致图谱断裂：

```
小学:
  自然数 → 加法 → 减法 → 乘法 → 除法
  分数 → 小数 → 百分数
  周长 → 面积 → 体积
  速度 → 路程 → 时间
  比例

初中:
  有理数 → 绝对值 → 相反数
  整式 → 单项式 → 多项式
  一元一次方程 → 二元一次方程组
  不等式 → 一元一次不等式
  函数 → 一次函数 → 正比例函数
  三角形 → 全等 → 相似
  勾股定理
  四边形 → 平行四边形 → 矩形 → 菱形 → 正方形
  实数 → 平方根 → 立方根
  二次根式
  一元二次方程
  二次函数
  圆 → 弧 → 弦 → 切线
  锐角三角函数 → sin → cos → tan
  数据的收集 → 平均数 → 中位数 → 众数 → 方差
```

### 运行管道

每提取完一批（如 Batch 1 的 4 个文件），运行：

```bash
python scripts/math_graph_pipeline/run_pipeline.py
```

检查输出中的 level 分布是否正确：
```bash
python -c "import json; d=json.load(open('data/math_extractions/merged/visualization_data.json')); \
from collections import Counter; \
print(Counter(n.get('level','?') for n in d['nodes']))"
```

如果 elementary/middle 出现，说明分类正确。
